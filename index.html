First Things First - What Is CSS And What Tools Do I Need

CSS, or Cascade Style Sheets, is a language that lets you style webpages; nothing more.

How To Use CSS

Selectors

          

           1 |  p {
           2 |  font-size: 10px;
           3 |  color: blue;
           4 |  }

Universal Selectors

The highlighted star is called the universal selector. This styles everything at once. People use this when they want to reset the whole HTML page, and start without any default values all browsers enable. 

          

           1 |  p {
           2 |  font-size: 10px;
           3 |  color: blue;
           4 |  }

Class Selectors

The dot selector is used to indicate you are targeting a class in your HTML to style. 

          

           1 |  p {
           2 |  font-size: 10px;
           3 |  color: blue;
           4 |  }

Id Selectors

The Hashtag/Pound symbol is used to indicate you are targeting an ID in your HTML to style. Note, you can only have one name for an ID in your HTML.

Linking Your CSS Files To Your HTML File:

          

           1 |  p {
           2 |  font-size: 10px;
           3 |  color: blue;
           4 |  }

Linking your CSS file is pretty simlpe as you copy the above line, then where it says "path to your css file," you put the path to your CSS file.
