Flexbox

First thing is first, the initial axis' need to be defined for flexbox, so hear goes. Being familir with the x,y plot system from algebra ( remember slopes? ), the two used in flexbox are: main axis ( x-axis ) and cross axis ( y-axis ). To master flexbox, knowing these when positioning items will be less confusing. for example, if the main flow of flexbox gets changed from row to column, the main axis and cross axis will switch places.

The flex shorthand sets the values for flex-grow, flex-shrink, and flex-basis all at once. When using flex, only one value number value is needed, and the rest is set automatically given the space for the flex item(s). Also, do not use flex-grow, flex-shrink, or flex-basis individually, instead use flex.
