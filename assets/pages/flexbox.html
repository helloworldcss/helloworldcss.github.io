How To Use CSS

Element Selector

          

           1 |  p {
           2 |  font-size: 10px;
           3 |  color: blue;
           4 |  }

The highlighted p selector is called an element selector. You can use the HTML element name as a CSS selector minus the . Say you wanted to select all the <a> tags in your HTML via CSS, you would use a as your selector.

Universal Selectors

          

           1 |  * {
           2 |  margin: 0;
           3 |  padding: 0;
           4 |  }

The highlighted * is called the universal selector. This styles everything at once. People use this when they want to reset the whole HTML page, and start without any default values all browsers enable.

Class Selectors

          

           1 |  .myClassName {
           2 |  font-size: 10px;
           3 |  color: blue;
           4 |  }

The . is used when you want to select a class in your HTML. Note, class names can be used as many times as you want in your HTML; meaning the class .highlight can be placed on as many HTML elements as you want.

Id Selectors

          

           1 |  #myClassName {
           2 |  font-size: 10px;
           3 |  color: blue;
           4 |  }

The # is used when you want to select an Id in your HTML. Note, Id names can only be used once in your HTML; meaning the Id #Name can only be placed on one HTML element.

Linking Your CSS Files To Your HTML File

          

           1 |  <link rel="stylesheet" href="./link/to/css/file.css">

Linking your CSS file is pretty simlpe, as you copy the above line. In the href="" attribute, you place the link to you CSS file.
